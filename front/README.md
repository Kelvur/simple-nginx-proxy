# Front 

This is a simple front made as example, you need NodeJS to compile

## How To Build

```bash
npm run build
```

### How To Build Pointing to Other Path

```bash
BASE_URL="mycustomserver.io:3000/api" npm run build
```


## Made With

- **OS**: Ubuntu 16
- **NodeJS**: v10.15
- **NPM**: 6.14
